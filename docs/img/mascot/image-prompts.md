# Image Prompts for Pemba the Red Panda

This file contains seven self-contained AI image generation prompts for **Pemba the Red Panda**, the pedagogical mascot for the Token Optimization textbook. Paste each prompt into ChatGPT/DALL-E, Midjourney, Stable Diffusion, or your preferred image generator. Generate at 512×512 or 1024×1024, save as PNG with transparent background, then trim padding.

After generating each image, save it to `docs/img/mascot/<filename>` and run:

```bash
python $BK_HOME/src/image-utils/trim-padding-from-image.py docs/img/mascot/<filename>
```

---

## 1. Neutral / Default Pose

Filename: **`neutral.png`**

```
I am about to ask you to generate 7 different poses of a mascot for a new book.
Please use a consistent drawing style for all 7 poses.

Please generate a new png image of Pemba the Red Panda in a neutral pose.

A modern flat vector cartoon illustration of Pemba, a friendly red panda
pedagogical mascot for a software engineering textbook on Token Optimization.
Pemba has classic red panda coloring: russet (deep red-brown) fur on top,
cream-colored belly and face markings, the signature white facial mask with
black "tear marks" below alert focused eyes, and a bushy tail with alternating
russet and cream rings. Pemba wears small wire-rim engineer's glasses. Friendly closed-mouth smile. The character is small and
compact, suitable for icon-sized display.

Pemba stands upright in a relaxed, neutral pose facing the viewer directly,
with a calm and friendly closed-mouth smile. Both paws rest naturally at the
sides with no specific gesture. The pose is balanced and unassuming —
suitable as a general-purpose or default illustration.

Style: modern flat vector, clean lines, fully transparent background,
suitable for embedding in educational content. No text in image.

The background MUST be fully transparent. DO NOT use a white, black, or
checkered background.
```

## After Generating All Images

1. Save each PNG into `docs/img/mascot/` using the filenames above (`neutral.png`, `welcome.png`, etc.)
2. Run the padding trimmer on each image so they display at the correct size in 90px admonitions:

   ```bash
   python $BK_HOME/src/image-utils/trim-padding-from-image.py docs/img/mascot/neutral.png
   python $BK_HOME/src/image-utils/trim-padding-from-image.py docs/img/mascot/welcome.png
   python $BK_HOME/src/image-utils/trim-padding-from-image.py docs/img/mascot/thinking.png
   python $BK_HOME/src/image-utils/trim-padding-from-image.py docs/img/mascot/tip.png
   python $BK_HOME/src/image-utils/trim-padding-from-image.py docs/img/mascot/warning.png
   python $BK_HOME/src/image-utils/trim-padding-from-image.py docs/img/mascot/encouraging.png
   python $BK_HOME/src/image-utils/trim-padding-from-image.py docs/img/mascot/celebration.png
   ```

3. Run `mkdocs serve` and visit the **Mascot Test** page (under Learning Graph) to preview all seven admonition styles together.
