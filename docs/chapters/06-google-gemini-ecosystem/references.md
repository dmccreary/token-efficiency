# References: The Google Gemini Ecosystem

1. [Gemini (language model)](https://en.wikipedia.org/wiki/Gemini_(language_model)) - Wikipedia - Coverage of the Gemini model family including Pro, Flash, and Ultra tiers and the long-context capabilities that distinguish Gemini from competitors.

2. [Google DeepMind](https://en.wikipedia.org/wiki/Google_DeepMind) - Wikipedia - Background on the research lab behind Gemini and its multimodal-first architecture decisions.

3. [Vertex AI](https://en.wikipedia.org/wiki/Vertex_AI) - Wikipedia - Overview of Google Cloud's managed ML platform and how Gemini API integrates with the broader GCP ecosystem.

4. AI Engineering - Chip Huyen - O'Reilly - Cross-vendor chapters on long context and grounding that contextualize Gemini's one-million-token window covered here.

5. Hands-On Large Language Models - Jay Alammar and Maarten Grootendorst - O'Reilly - Chapters on multimodal input and long-context retrieval map directly to Gemini's distinctive capabilities.

6. [Google Gemini API Documentation](https://ai.google.dev/gemini-api/docs) - Google - Authoritative reference for the Gemini API including model identifiers, request format, and response schema used throughout this chapter.

7. [Vertex AI Generative AI Documentation](https://cloud.google.com/vertex-ai/generative-ai/docs) - Google Cloud - Reference for the enterprise deployment path including IAM, regional endpoints, and quota management.

8. [Google AI Studio](https://aistudio.google.com/) - Google - Browser-based prompt experimentation environment for Gemini models; useful for prompt-engineering iteration before committing to production code.

9. [Gemini Long Context Guide](https://ai.google.dev/gemini-api/docs/long-context) - Google - Reference for techniques and limitations of the 1M-token context window including the lost-in-the-middle effect and recommended chunk-ordering patterns.

10. [Gemini Context Caching Documentation](https://ai.google.dev/gemini-api/docs/caching) - Google - Reference for Gemini's caching API including minimum cache size, TTL configuration, and pricing — Google's analogue to Anthropic's prompt caching covered in Chapter 14.
