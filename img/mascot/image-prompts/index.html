    Image Prompts for Pemba the Red Panda

    This file contains seven self-contained AI image generation prompts for Pemba the Red Panda, the pedagogical mascot for the Token Optimization textbook. Paste each prompt into ChatGPT/DALL-E, Midjourney, Stable Diffusion, or your preferred image generator. Generate at 512×512 or 1024×1024, save as PNG with transparent background, then trim padding.

    After generating each image, save it to docs/img/mascot/<filename> and run:

    1
    python $BK_HOME/src/image-utils/trim-padding-from-image.py docs/img/mascot/<filename>
